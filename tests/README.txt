Test case usage:

1. You need Stagehand_TestRunner(Piece Project) and simpletest.

- Stagehand_TestRunner : http://trac.piece-framework.com/sh-testrunner/wiki/en/Start
-- note: version >= 1.2.0 needed.

- simpletest : http://www.lastcraft.com/simple_test.php

2. Let's test!

$ testrunner-st -p prepare.php xxxx_TestCase.php

or, all test, 
$ testrunner-st -p prepare.php -R

Thank you.
